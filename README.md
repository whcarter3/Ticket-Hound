# Ticket-Hound.us

A simple source for finding the cheapest tickets to concerts near you.


###Ticket Hound was built using the following technologies:###

* Ruby
* Rails
* Angular.js
* Mapbox.js
* A number of ticket provider APIs
* The blood and sweat of two developers.
